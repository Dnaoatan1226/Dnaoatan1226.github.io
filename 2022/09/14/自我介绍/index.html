我是2020级计算机类三班的黄霆楠，现属计算机应用方向。

3.学习经历

上大学之前本来想学的专业是物理方面的，但是鉴于计算机专业就业好，还是选择了计算机专业。前两年的大学时间重心都在卷绩点和打球上，说是卷绩点但平时还是比较摆，属于是期末努力型。平时除了完成老师布置的代码作业外，打代码的时间比较少。自己也有想多学学一些专业知识，但总是学了一会会就半途而废，所以现在专业知识欠缺。

专业相关

1.已掌握的知识

  • c/c++
  • 基础的算法与数据结构
  • java基础语法

2.感兴趣的方向

  • 后端
  • 游戏开发

3.正在学习的知识

  • java
  • python

4.不足的地方

  • 比较社恐
  • 没有一门精通的语言
  • 做事经常三分钟热度
  • 没有项目开发的经验
  • 专业知识的学习不够深入

知道很菜了，会好好学习提高编程能力的。

未来规划

  • 大三学年，上学期好好卷绩点，争取保研，如果失败了，大三下学期就好好的准备考研。
  • 大四上学期考研冲刺，下学期好好做毕业设计。
  • 读完研究生出去找工作当个社畜咯。

最后，希望通过这门课程好好锻炼自己的代码能力。
